회의내용

■ 회의 목적: 기존 설계시의 문제점을 수정 및 보완

■ 나온 안건

  • 입력할 사용자 데이터의 부족으로 자세한 사용자와 평균 통계의 비교가 불가능한 문제를 해결하기 위해 데이터 베이스에 저장할 사용자 데이터의 양을 늘린다.
  • 사용자의 데이터와 질병의 통계를 비교 분석하는 기능은 구현이 어려움으로 전체 통계를 더 자세히 보여주는 방향으로 목표를 바꾼다.
  • 다국어 지원을 전부 구현하기에 시간적으로나 정확한 번역 확인이 어렵기 때문에 언어는 한국어와 영어로만 줄인다.
  • 기존 통계자료에서 구현이 어려운 질병을 제외하고 성별과 연령별로만 통계자료를 구성한다.

■ 최종 대쉬보드 수정 사항

  • 메인 페이지
    • 연령대별 건강검진 참여도
    • 성별 건강검진 참여도
    • 건강검진 시 알면 좋은 점이나 주의사항 등을 제공
  • 사이드바
    • 건강 검진으로 진단 가능한 질병들을 관련이 있는 질병으로 묶어 리스트화
  • 로그인 화면
    • 사이드바에서 로그인 하는거 대신에 로그인 화면에서 로그인해야 대시보드에 접근할 수 있도록 함
  • 질병별 페이지
    • 성별과 연령별에 따른 질병 통계를 표시
    • 각 질병에 대한 설명을 제공
    • 예방 방법과 전문가의 의견을 유튜브 링크 등으로 제공
